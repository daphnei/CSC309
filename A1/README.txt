Instructions
------------
1. Open unix terminal
2. Run server 'node index.js'
3. Access site using the URL: http://localhost:31285/
4. [Optional] Populate frontpage by running every command from top to bottom in the Populate.txt file, or by piping the file's commands into the shell with either "bash < Populate.txt" or "tcsh < Populate.txt".
